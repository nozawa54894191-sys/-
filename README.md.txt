# 最低賃金チェック（時給換算 + 固定残業代補助）

都道府県別最低賃金に対して**時給換算で自動判定**するシンプルなWebツールです。  
月給の場合は「勤務時間方式（総時間 / 勤務日数+実働）」を選択後に必要な入力欄のみ表示し、**固定残業〈時給〉が最低賃金×1.25**を満たすかを判定します。  
ページ下部に **2025年 地域別最低賃金一覧（厚労省順）** を掲載。

## 使い方
1. このリポジトリをCloneまたはZIPダウンロード
2. `index.html` をブラウザで開くだけ

## GitHub Pagesで公開
1. このリポジトリをGitHubにPush  
2. **Settings → Pages** で Source を「Deploy from a branch」、Branch を `main` / `/ (root)` に設定  
3. 数十秒後に公開URLが発行されます

## 入力仕様
- 都道府県（厚労省公表順で選択）
- 給与形態：時給 / 日給 / 月給
- 給与額
- 勤務時間形態（**月給のみ**）
  - 総労働時間 もしくは 勤務日数 + 1日の実働時間
- 固定残業代（任意・月給向け）
  - 金額（円/月）、想定時間（h/月）

## 判定ロジック（概要）
- **本体の最低賃金比較**：  
  - 時給・日給は単純換算  
  - 月給は **（月給 − 固定残業代） ÷ 所定時間**（=基礎賃金）で比較
- **固定残業の妥当性チェック**：  
  - 固定残業〈時給〉＝ **固定残業代 ÷ 固定残業時間**  
  - 判定基準〈時給〉＝ **地域別最低賃金 × 1.25**  
  - 固定残業〈時給〉が基準以上かを OK/NG 表示
- **改定前表示**：本日が改定日前の場合、判定結果に「改定日」と「改定後額」を併記

## データ更新（毎年）
- 2025年の配列 `MW2025` を `assets/app.js` 内で定義しています。  
  次年度は新しい配列に差し替えてください（`new`/`prev`/`effective`）。

## 免責
- 本ツールは地域別最低賃金の**目安確認**を目的としています。  
  **特定（産業別）最低賃金**が適用される場合や、最終判断は労務担当・所轄労基署の指示に従ってください。
